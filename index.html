Twitch TV

Twitch TV

This project was built as a part of Free Code Camp's Intermediate Front-End Development curriculum.

Objective:

Build a CodePen.io app that successfully reverse-engineers this.

User stories:

  • As a user, I can see whether Free Code Camp is currently streaming on Twitch.tv.
  • As a user, I can click the status output and be sent directly to the Free Code Camp's Twitch.tv channel.
  • As a user, if Free Code Camp is streaming, I can see additional details about what they are streaming. As a user, I can search through the streams listed.
  • If a streamer has closed his/her Twitch account, it will not be displayed to the user.

See the PenTwitch TV Zipline for Free Code Camp by Veronika Benkeser (@veronikabenkeser) on CodePen.
